### DNAnalyzerX

[](https://opensource.org/licenses/MIT)  
[](https://www.python.org/)

DNAnalyzerX - программное обеспечение для анализа точечных участков ДНК человека на предмет наличия мутаций, а также определения сопровождающих их заболеваний.

* * *

### Функционал

Здесь перечисляются ключевые возможности и особенности проекта:

*   **Предназначеие** - эффективное сравнение отрезков ДНК на основе реальных алгоритмов биоинформатики (Смит-Ватерман).
*   **Оптимизация** - Использование numba и С-оптимизация математических функций.
*   **Интерфейс** - Легкий и интуитивный интерфейс на CustomTk.
*   **Экспорт анализа** - Возможность сохранить результат анализа.

* * *

### Инструкция по запуску

Следуйте этим шагам, чтобы развернуть проект локально.

### Установка

1.  **Клонируйте репозиторий:**
    
    bash
    
        git clone https://github.com/LoggysGit/DNAnalyserX
        cd DNAnalyserX
        
    
2.  **Настройте виртуальное окружение:**
    
    bash

        python -m venv .venv
        
    
3.  **Установите зависимости:** 
    bash

        pip install -r requirements.txt

### Запуск приложения

bash

    # Режим разработки
    npm run dev
    
    # Производственный режим
    npm start
    

Use code with caution.
